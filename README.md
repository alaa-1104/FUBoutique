# For U Boutique Website

A responsive e-commerce website built with HTML, CSS, and JavaScript, deployed using GitHub Pages,
the website allows users to browse different clothing categories and proceed to a checkout page
to simulate the purchasing process.

## Project Description
The website is designed as an official boutique website where users can:
- Browse clothing categories (Men, Women, Kids)
- Navigate between multiple pages
- Add items and proceed to a checkout page
- View store information and contact details

The project focuses on front-end web development using HTML, CSS, JavaScript, and Bootstrap.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 4
- Font Awesome

## Website Pages
- Home page
- Men products page
- Women products page
- Kids products page
- Checkout page
- Contact page

## Project Structure
- index.html
- Men.html
- Women.html
- Kids.html
- checkout.html
- css/
- js/
- images/
- bootstrap files

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Navigate through the website using the menu links.

## Notes
This is a front-end static project created for learning and demonstration purposes.
No backend or payment integration is implemented.
